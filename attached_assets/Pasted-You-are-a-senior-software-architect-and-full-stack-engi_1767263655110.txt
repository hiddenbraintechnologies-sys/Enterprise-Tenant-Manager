You are a senior software architect and full-stack engineer.

Build a scalable, production-ready, multi-tenant SaaS platform that supports multiple business types including:

• PG / Hostels
• Salons & Beauty Parlours
• Gyms & Fitness Centers
• Clinics & Hospitals
• Diagnostic Centers
• Service-based small businesses

The system must be enterprise-ready, modular, secure, and future-proof.

===================================
CORE ARCHITECTURE REQUIREMENTS
===================================

- Use Node.js with Express (upgrade-friendly to NestJS)
- PostgreSQL database
- ORM: Drizzle or Prisma
- REST-based API architecture
- JWT-based authentication
- Role-based access control
- Multi-tenant isolation
- Docker-ready architecture
- Cloud-ready (AWS compatible)

===================================
MULTI-TENANT DESIGN
===================================

Each tenant must have:
- Unique tenant_id
- Configurable enabled modules
- Custom branding (logo, theme, domain)
- Isolated data access
- Feature flags

All tables must include:
- tenant_id
- created_at
- updated_at
- soft_delete support

Tenant context must be enforced via middleware.
